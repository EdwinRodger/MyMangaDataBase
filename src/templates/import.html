MyAnimeList Import

Imports MyAnimeList " .xml " file.

Follow the steps below before importing manga list from MyAnimeList -

  1. Extract .xml file from .gz file
  2. Open .xml file
  3. Delete the first line from xml file which is < ?xml version="1.0" encoding="UTF-8" ?>
  4. Save the file
  5. Import it in MMDB
NOTE: Importing the file as is will give a server error and the file won't get imported

MangaUpdates Import

Imports MangaUpdates " {status}_YY_MM_DD.txt " file

The {status} from MU file name will be applied to the imported manga from that list

NOTE: Doesn't supports custom lists
