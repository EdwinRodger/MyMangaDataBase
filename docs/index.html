MyMangaDataBase

A personal, simple and strightforward manga database to keep track of all your Anime and Manga. Your list, Your control! Don't wait for someone to add anime/manga to the site's database, make your own database.
You can see the demo at https://mymangadatabase.pythonanywhere.com/

Features

FOSS

MyMangaDataBase is Free and Open Source Software built in Python as backend and HTML with Bootstrap5 as frontend. You can find the source code in the github repository.

Self hosted

MyMangaDataBase is fully self hosted option. It uses waitress as its hosting server.

Responsive UI

As MyMangaDataBase is made in Bootstrap5 which is known to be a mobile first CSS framework, MMDB has decent mobile user interface out of the box.
